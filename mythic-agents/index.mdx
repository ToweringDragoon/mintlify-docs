## Overview

This documentation covers a comprehensive collection of Mythic agents designed for post-exploitation operations, system integration, and command augmentation. Mythic is a collaborative, multi-platform, red teaming framework that provides a flexible and extensible architecture for command and control operations.

The agents documented here span multiple categories:
- **Payload Agents**: Traditional post-exploitation agents running on compromised systems
- **Service & Integration Agents**: Specialized agents for platform integration and data enrichment
- **Command Augmentation**: Tools that extend capabilities across multiple agent types

All agents support the Mythic framework's core features including task management, file operations, credential tracking, and comprehensive logging.
