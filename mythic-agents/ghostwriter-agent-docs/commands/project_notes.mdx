## Summary

Retrieve all notes associated with the project for team collaboration.

- Needs Admin: False
- Version: 1-2
- Author: @its_a_feature_

### Arguments

No arguments required.

## Usage

```
project_notes
```

## Detailed Summary

Retrieves all project notes including:

### Note Information
- Note title and content
- Author
- Creation and modification timestamps
- Tags/categories

Project notes are used for team collaboration, documenting important information that doesn't fit into findings or oplogs.
