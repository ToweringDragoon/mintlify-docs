## Summary

Retrieve generated report files and templates for the project.

- Needs Admin: False
- Version: 1-2
- Author: @its_a_feature_

### Arguments

#### report_id
- Description: ID of the report to get artifacts for
- Required: False


## Usage

```
reports_artifacts
reports_artifacts -report_id 3
```

## Detailed Summary

Retrieves report artifacts including:

### Artifacts
- Generated report files (Word, Excel, etc.)
- Report templates used
- Supporting documents
- Appendices

Allows downloading of finalized reports and associated documents generated by Ghostwriter.
