## Summary

Retrieve information about all reports associated with the project.

- Needs Admin: False
- Version: 1-2
- Author: @its_a_feature_

### Arguments

No arguments required.

## Usage

```
reports_get
```

## Detailed Summary

Retrieves all report information for the project including:

### Report Details
- Report title and type
- Creation and delivery dates
- Status (draft, in review, delivered)
- Associated findings count
- Evidence count
- Assigned report writers

Reports are the final deliverables that contain findings, evidence, and analysis.
