## Summary

Get information about the current project. This command queries Ghostwriter for details about the active engagement project.

- Needs Admin: False
- Version: 1-2
- Author: @its_a_feature_

### Arguments

No arguments required.

## Usage

```
project
```

## Detailed Summary

This command retrieves comprehensive project information including:

### Project Details
- Project codename and name
- Start and end dates
- Project type
- Client information (name, codename, timezone)

### Team Information
- Project lead
- Team members and their roles
- Operator assignments

All project data is scoped to the current Mythic operation and automatically mapped to the corresponding Ghostwriter project.
