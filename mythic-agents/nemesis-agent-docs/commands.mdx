## Nemesis Command Reference

These pages provide documentation for the `Nemesis` integration commands that allow you to interact with the Nemesis data processing platform.

## Available Commands

- [upload](./commands/upload) - Upload files to Nemesis for processing
- [chromium](./commands/chromium) - Get Chromium browser information from Nemesis
- [hashes](./commands/hashes) - Get extracted hash information from Nemesis
- [credentials](./commands/credentials) - Get credential information from Nemesis
- [triage](./commands/triage) - Get file triage status information from Nemesis
