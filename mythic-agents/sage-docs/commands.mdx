## Sage Command Reference

These pages provide documentation for the Sage agent commands that allow you to interact with AI and LLM services through the Mythic framework.

## Available Commands

### AI Interaction
- [chat](./commands/chat) - Interactive chat session with AI models
- [query](./commands/query) - Send a single query and receive a single response
- [list](./commands/list) - List available models from a provider

### Extensions
- [mcp-connect](./commands/mcp-connect) - Connect to a Stdio Model Context Protocol server
