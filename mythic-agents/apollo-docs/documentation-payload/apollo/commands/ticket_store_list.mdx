## Summary
list information about all loaded tickets in the agents ticket store.


### Arguments


#### luid
Optional argument to filter the tickets in the agents store to ones matching a specified luid.



## Usage
```
ticket_store_list -luid [luidValue]
```

Example
```
ticket_store_list -luid [luidValue]
```
